  <name>Task 2: Implement secure file ID generation</name>
  <files>server/src/utils/fileUtils.ts</files>
  <action>Create file utility functions:

1. generateFileId(): string
   - Use crypto.randomUUID() for UUID v4
   - Returns standard UUID format (e.g., "7a3f8d2e-...")

2. getFileDirectory(fileId: string): string
   - Takes first 2 chars of UUID as subdirectory name
   - Returns path like "uploads/videos/7a/"
   - Prevents single directory with thousands of files

3. getFilePath(fileId: string, extension?: string): string
   - Returns full path to file
   - Optionally append extension

4. sanitizeFilename(filename: string): string
   - Remove path traversal attempts (../, ..\)
   - Keep only safe characters
   - Limit length

5. ensureDirectory(dirPath: string): Promise<void>
   - Create directory if not exists
   - Handle nested directory creation

6. getFileExtension(filename: string): string
   - Extract extension safely
   - Return lowercase

All functions must prevent path traversal attacks.</action>
  <verify>Write tests for:
- generateFileId returns valid UUID
- getFileDirectory returns correct subdirectory
- sanitizeFilename blocks ../ attacks
- Path traversal attempts are rejected</verify>
  <done>Utility functions implemented, security tests pass</done>
</task>

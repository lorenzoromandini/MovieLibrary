  <name>Task 2: Create authentication middleware</name>
  <files>server/src/middleware/auth.ts</files>
  <action>Create Express authentication middleware:

1. requireAuth middleware:
   - Extract token from httpOnly cookie
   - Verify JWT token
   - Attach user info to req.user
   - Call next() if valid
   - Return 401 if invalid/missing

2. optionalAuth middleware:
   - Similar to requireAuth but doesn't fail if missing
   - Attach user if present, continue either way
   - Useful for routes that work both authenticated and not

3. requireRole middleware factory:
   - requireRole(role: string) returns middleware
   - Check if req.user.role matches
   - Return 403 if not authorized

4. Error handling:
   - Clear cookie on auth failure
   - Return appropriate status codes
   - JSON error responses

5. TypeScript declaration:
   - Extend Express.Request to include user property

Use the auth config for token verification.</action>
  <verify>Test middleware with mock requests, verify 401/403 responses</verify>
  <done>Auth middleware implemented, tested with valid/invalid tokens</done>
</task>

# 🎬 MovieLibrary

**A private movie collection for friends and family.**

This is a personal movie library service where you can browse and download movies from my collection. Access is by invitation only.

---

## 🔐 How to Get Access

This service is private and invitation-only. Here's how it works:

### 1. Request Access
Contact me (the admin) to request access to the library. You can reach me via:
- Email
- WhatsApp/Telegram
- In person

### 2. Receive Your Access Code
Once I approve your request, I'll send you a unique access code (a one-time token) via your preferred contact method.

### 3. Enter Your Code
1. Visit the website
2. Click "Request Access" on the home page
3. Enter the access code I sent you
4. You're in! You can now browse and download

**Important:** Each access code can only be used once and is tied to your device/browser session.

---

## 🎥 What You Can Do

Once authenticated, you can:

### Browse the Collection
- View all movies in a visual poster grid
- Movies are sorted by recently added by default
- Scroll through the entire collection

### Search & Filter
- **Search by title** — Type to find movies (supports both original and Italian titles)
- **Filter by year** — Select a specific year or range
- **Filter by genre** — Select one or more genres
- **Search by actor/director** — Find movies with your favorite actors

### View Movie Details
Click on any poster to see:
- Large movie poster
- Original and Italian titles
- Release year
- Director and main cast
- Runtime
- Genres
- Plot summary

### Download Movies
- Click the **Download** button on any movie detail page
- Files download directly to your device (no streaming)
- Supported formats: MP4, MKV, AVI, MOV, WebM

---

## 💡 Tips for Using the Service

### Finding Movies
- Use the search bar at the top for quick title searches
- Combine filters: try "Action movies from 2020-2023"
- Don't remember the exact title? Try partial words — the search is fuzzy!

### Download Quality
- All movies are available in their original quality
- File sizes range from 1GB to 50GB depending on the movie
- Make sure you have enough storage space before downloading

### Browser Compatibility
- Works best in Chrome, Firefox, Safari, or Edge
- Enable downloads for the site if prompted
- Large downloads work best on stable Wi-Fi connections
